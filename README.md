# About
This repo contain the code and data used in my thesis: [Characterization of design discussions in Modern Code Review](http://dspace.sti.ufcg.edu.br:8080/xmlui/handle/riufcg/21155)

- design-classifier: binary classifier for identifying design discussions in MCR discussions
- gt-study: helper scripts used in the grounded theory study
- preprocess-dataset: a script that prepares the data for the binary classifier of design discussions
- rbminer: the Python scripts for mining data from the public Apache ReviewBoard
- training-data: annotated training data used to train the classifier
